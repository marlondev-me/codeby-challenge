## Install

You need the [Node.js](https://nodejs.org/en/download/) and [Yarn](https://classic.yarnpkg.com/en/docs/install/) to run this project, this example was created with `Node.js v14.16.0` and `Yarn v1.22.5`.

### Development Mode

```bash
yarn install
yarn dev
```

### Production Mode

```bash
yarn install
yarn build
yarn start
```
